# Enable Centos bash completion
1. install following packages 
```bash
yum install bash-completion bash-completion-extras
```
1. source ``` bash_compeletion.sh ```
```bash
locate bash_completion.sh
source /etc/profile.d/bash_completion.sh  
```
1.re-login
